cbonsai screen saver option reference

These settings are compiled into command-line arguments and passed to the bundled original cbonsai executable when the screen saver starts. The saver always passes --live and --infinite.

Timing

Tree growth interval --time=TIME

Sets the delay between growth steps. Smaller values animate faster; larger values animate slower.

Growth restart wait time --wait=TIME

Sets how long to wait after a tree finishes before starting the next growth cycle.

Tree

--message=STR

Sets a message for cbonsai to render with the tree. Empty messages are omitted from the command line.

Pot style --base=INT

Chooses style 1, style 2, or no pot. The saver passes the selected value as --base=1, --base=2, or --base=0.

Leaf character --leaf=LIST

Sets the character used for leaves. The value is passed through to cbonsai as the leaf list.

Tree colour --color=LIST

Chooses fixed ANSI colours for dark leaves, dark wood, light leaves, and light wood. The saver passes the selected colours as cbonsai's comma-separated colour list.

Tree density --multiplier=INT

Controls branch density and growth scale. Larger values generally create fuller trees.

Branch lifetime duration --life=INT

Controls how long branches continue growing. Larger values generally create taller or more complex trees.

Output

--seed=INT

Uses a fixed random seed. Enable this when you want deterministic tree output for the same settings.

--save=FILE

Saves cbonsai output state to a file. The saver only passes this argument when Save file is enabled and the path is not empty.

--load=FILE

Loads cbonsai output state from a file. The saver only passes this argument when Load file is enabled and the path is not empty.

--verbose

Requests more detailed cbonsai output.

Bundled with cbonsai screen saver for offline option help.
