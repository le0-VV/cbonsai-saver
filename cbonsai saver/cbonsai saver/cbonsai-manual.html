cbonsai screen saver option reference

These settings are compiled into command-line arguments and passed to the bundled original cbonsai executable when the screen saver starts.

General

Font size

Terminal font size used by the screen saver renderer. This is a screen saver display setting and is not passed to cbonsai.

Mode

--screensaver

Runs cbonsai in screen saver mode so it continuously draws new trees without interactive input.

--live

Shows the tree as it grows instead of only drawing the completed result.

--infinite

Keeps growing trees indefinitely. Use this when you want cbonsai itself to keep running rather than exiting after one tree.

Timing

--time=TIME

Sets the delay used while the tree grows. Smaller values animate faster; larger values animate slower.

--wait=TIME

Sets how long to wait after a tree finishes before starting or exiting, depending on the selected mode.

Tree

--message=STR

Sets a message for cbonsai to render with the tree. Empty messages are omitted from the command line.

--base=INT

Sets the tree base style. The saver only passes this argument when the Base checkbox is enabled.

--leaf=LIST

Sets the character list used for leaves. The value is passed as a comma-separated list.

--color=LIST

Sets the ANSI color list used for leaves. The value is passed as a comma-separated list.

--multiplier=INT

Controls branch density and growth scale. Larger values generally create fuller trees.

--life=INT

Controls how long branches continue growing. Larger values generally create taller or more complex trees.

Output

--print

Prints the completed tree when cbonsai finishes. This is usually less useful for a continuous screen saver, but it is exposed for parity with cbonsai.

--seed=INT

Uses a fixed random seed. Enable this when you want deterministic tree output for the same settings.

--save=FILE

Saves cbonsai output state to a file. The saver only passes this argument when Save file is enabled and the path is not empty.

--load=FILE

Loads cbonsai output state from a file. The saver only passes this argument when Load file is enabled and the path is not empty.

--verbose

Requests more detailed cbonsai output.

--help

Prints cbonsai's command-line help and exits. This is useful for checking the installed binary's supported options, but it will not act like a running screen saver.

Bundled with cbonsai screen saver for offline option help.
